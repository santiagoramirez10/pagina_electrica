¿Qué es Python

Es un lenguaje de programación de propósito general, y uno de los más usados.

  • Lenguaje de alto nivel.
  • Interpretado.
  • Orientado a objetos (OOP).
  • Sintaxis sencilla y fácil de leer.
  • Creado por Guido Van Rossum (Holanda) en 1989 (Monty Python).
  • Dos versiones vigentes: 2.7 y 3.x (3.11.1).
  • Aplicaciones científicas:
    • Cálculos numéricos.
    • Representación gráfica de datos.
    • Análisis estadístico.
    • Simulación y optimización.
    • Inteligencia artificial.
  • Bases de datos.
  • Aplicaciones web y móviles.
  • Páginas web.
  • Videojuegos.

Características principales

  • Lenguaje indentado (pseudocódigo)
  • Fácil de aprender
  • Código fácil de leer
  • De código abierto → Sin costo por su uso
  • Versátil → Todo tipo de aplicaciones.
  • Soporte → Gran comunidad de usuarios (ayuda y librerías)
  • Multiplataforma → Windows, Linux, Mac
  • Embebido y extensible → Compatibilidad con otros lenguajes.
  • Interpretado → Velocidad de desarrollo y depuración
  • OOP → Seguridad y economía de código.

Entornos de desarrollo

Instalación local

En la nube: Google Colab (colaboratory) → https://colab.research.google.com/ Creación y ejecución de programas de Python en un navegador de internet.

  • Sin necesidad de instalación ni configuración
  • Acceso gratuito a GPUs
  • A través de internet → Cualquier dispositivo (móviles, tabletas)
  • Respaldo de la información (Google drive)
  • Facilidad de compartir archivos
  • Trabajo colaborativo.
  • Colab notebook → Similar a Jupyter Notebook

Cuadernos de Jupyter

  • Ejecución en un navegador de internet (instalación local o en la nube)
  • Pestaña navegador de archivos
  • Una pestaña por cada programa de Python.
  • Componente básico: celdas → Ejecución individual
  • Celdas de código → Código de Python (comentarios: #)
  • Celdas de texto / gráficos → Markdown
  • Resultados (datos y gráficas) → Salida de la celda correspondiente
